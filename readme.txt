A collection of 25 modern, visually attractive CSS preloaders with easy admin controls and site-wide implementation.

== Description ==

Modern Preloaders provides 25 contemporary CSS loading animations that can be easily integrated into any WordPress website. Each preloader features modern web animation techniques including smooth rotations, pulsing effects, bouncing animations, morphing shapes, and gradient effects.

**Features:**

* 25 unique, modern CSS preloader animations
* Easy-to-use admin interface under Settings > Preloader Settings
* Enable/disable preloader functionality site-wide
* Live preview of all preloaders in the admin area
* Responsive grid layout for easy selection
* Smooth fade-out transition when pages finish loading
* Lightweight and optimized for performance
* Compatible with all modern browsers
* No external dependencies

**Preloader Types:**

* Rotating rings and spinners
* Pulsing dots and circles
* Bouncing ball animations
* Morphing geometric shapes
* Gradient effects
* Wave animations
* Folding cubes
* DNA helix patterns
* Heartbeat effects
* Clock animations
* And many more!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/modern-preloaders` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Preloader Settings to configure the plugin.
4. Enable the preloader and select your preferred animation.
5. Save the settings and view your website to see the preloader in action.

== Frequently Asked Questions ==

= How do I enable the preloader? =

Go to Settings > Preloader Settings in your WordPress admin area, check the "Enable preloader site-wide" checkbox, select your preferred preloader animation, and click "Save Changes".

= Will the preloader affect my website's performance? =

No, the preloaders are lightweight CSS animations that are only loaded when enabled. The JavaScript for hiding the preloader is minimal and optimized.

= Can I customize the preloader colors? =

The current version includes 25 pre-designed preloaders with carefully chosen color schemes. Custom color options may be added in future versions.

= Does it work with caching plugins? =

Yes, the plugin is compatible with all major caching plugins as it uses standard WordPress hooks and functions.

= What browsers are supported? =

The preloaders work in all modern browsers that support CSS3 animations, including Chrome, Firefox, Safari, and Edge.
